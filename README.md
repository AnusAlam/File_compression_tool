## Compress your text files using Huffman Encoding
A python based file compression tool using Huffman Encoding and the principles of Data Structures and Algorithms.

## must do
Leave the first line of your text file blank and start your contents from second line. Because, the program has some kind of bug, therefore it usually fails to decode the first few words of the text file correctly, this could loss your important data but, if you start the file from second line then your data will be decoded correctly and this ensures the safety of your data.
You can test this on "sample.txt" to get the idea!

## Compression achieved
Upto 40%

## Usage
python path/to/file.py
